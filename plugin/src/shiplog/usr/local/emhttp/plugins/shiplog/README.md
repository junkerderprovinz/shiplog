**ShipLog**

Read-only update advisor, right inside Unraid's Docker tab. Next to each container ShipLog shows what changes between your running image and the newest one: the release notes, a deterministic risk badge (patch / minor / major) and the real version jump (e.g. 1.7 &rarr; 1.8) &mdash; before you press update.

Read-only &mdash; it never pulls, recreates or stops anything. Optional AI changelog summaries via a local Ollama, and Matrix notifications.
